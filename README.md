# Task-04: Twitter Sentiment Analysis

This project analyzes and visualizes sentiment patterns in Twitter data to understand public opinion toward different brands and topics.

## Dataset
- twitter_training.csv (Kaggle)

## Steps Performed
- Data loading and cleaning
- Sentiment distribution analysis
- Visualization using bar charts
- Brand-wise sentiment analysis

## Tools Used
- Python
- Pandas
- Matplotlib

## Outcome
The analysis provides insights into positive, negative, neutral, and irrelevant sentiments expressed by users on Twitter.
